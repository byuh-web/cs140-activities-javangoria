Questions

  1. List out the different ways to select an item

    1. You can select basic selectors
    2. You can select class selectors

  2. What is the difference between an ID and a Class?

    3. ID is unique and can only be used once per page, while class can be used multiple times.

  3. What is the difference between a SPAN tag and a DIV tag?

    4. SPAN is an inline element, while DIV is a block-level element.

  4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

    5. HEADER, MAIN, and FOOTER are semantic elements that define the structure of a webpage, while DIV is a generic container.

  5. What are the different ways to select a color?

    6. You can select colors using color names, HEX codes, RGB values, RGBA values, HSL values, and HSLA values.
