Questions

  1. Explain the difference inline, embedded and external CSS.

Inline CSS is applied directly to an HTML element using the style attribute. This method is specific to individual elements.

For Example: <p style="color: blue; font-size: 16px;">Good Morning</p>

Embedded CSS is defined within a tag inside the <style> section of an HTML document. This applies styles to the entire document

For Example:

<style>
  p { color: lightblue; }

</style>

External CSS is defined in a separate .css file and linked to the HTML document using the <link> tag in the <head> section. This method allows for consistent styling across multiple pages.

For Example:

<link rel="stylesheet" href="styles.css">
  1. Explain why you should only use external CSS.

External CSS is considered the best practice because it separates content from presentation, making it easier to maintain and update styles across multiple pages. It also improves performance by allowing browsers to cache the CSS file, reducing load times on subsequent visits.

  1. What is a selector used for?

    2. A selector is used in CSS to target HTML elements that you want to style. It defines which elements the CSS rules will apply to, allowing you to customize the appearance of specific parts of your web page.
